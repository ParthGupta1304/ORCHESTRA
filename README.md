# 🎼 Orchestra — Multi-Agent Hackathon Judging System

Orchestra is a full-stack system that deploys a panel of specialist AI agents to evaluate hackathon submissions across 5 key dimensions:
- Innovation
- Technical Implementation
- Business Viability
- Presentation Quality
- Clarity & Problem Definition

It produces objective scores, a ranked leaderboard, and personalised feedback reports.

## Project Structure
- **/backend**: Express.js server, SQLite database, and specialist AI agents powered by GPT-4o.
- **/frontend**: Next.js (TypeScript + Tailwind CSS) dashboard for submissions and leaderboard.

## Quick Start
1. Configure `.env` in the backend.
2. Run `npm install` in both directories.
3. Start the judging pipeline!
